pdf2md·pro

Trascrizione PDF → Markdown per contenuti tecnico-accademici. Tabelle, formule, figure descritte, provenienza per pagina.

01 · Conversione

Motore

Il modello GLM-OCR si usa in Ibrido e LLM totale. Scegli sotto quale motore LLM: GLM-OCR locale (default) o OpenRouter.

Cartelle
tutti i PDF della cartella
Partizionamento automatico

Ogni file .md mantiene il nome del PDF originale (salvo l'opzione "Rinomina per argomento"). Ogni conversione produce anche il sidecar .provenance.json. Le impostazioni vengono salvate automaticamente e ricaricate al prossimo avvio (la chiave API solo se spunti "Ricorda la chiave").

02 · Partizionatore PDF

Spezza i PDF fuori limite

L'analisi non scrive nulla: elenca ogni PDF con pagine, MB e limiti superati. La partizione lavora in loco: le parti nome 03 01.pdf (totale sezioni + progressivo) restano nella cartella, e ogni originale spezzato viene spostato nella sottocartella interi/. Il passo 3 converte la stessa cartella in Markdown (motore e destinazione della sezione 01); interi/ viene ignorata.
